# Atomly-core-v1 盲测数据说明（供转发）

本压缩包含 100 条合成粉末 XRD，用于“各方法原生数据库/检索空间”的端到端盲测。
不包含候选 CIF、真实相标签、比例或逐样本相数。

## 请提供给测试人员的信息

- `patterns/`：100 个两列 `.xy` 文件，第一列为 2theta（度），第二列为 counts。
- `sample_manifest.csv`：每个样本的元素集合、文件名、网格和校验哈希。
- `prediction_template.csv`：建议按该表返回，每个预测相一行。
- `instrument_metadata/`：对需要仪器 profile 的方法公开的合成测量元数据。

## 统一测试协议

1. 主测试只公开盲谱、精确元素集合和测量元数据；不公开输入 CIF 或逐样本二相/三相标签。
2. 四种方法均使用固定的全局上限 3，相数由各自停止规则/模型比较决定。
3. MatDiffract、XERUS、Dara 使用各自冻结的数据库流程；CrystalShift 必须明确标注其外接的数据库前端。
4. 若方法能够物理定量，请返回质量分数并让同一候选解内比例和为 1。
5. CrystalShift activation 未经物理校准时比例列留空，不得改名为摩尔或质量分数。
6. 请返回方法原生数据库 ID；最好同时保存入选 CIF，便于结构级映射，但这些 CIF 是输出而非输入。
7. 请返回每个样本总运行时间；未成功的样本也保留一行并填写 `status_or_note`。

## 数据范围

- 50 个二相、50 个三相样本；逐样本相数隐藏；质量分数为连续随机数，不限制为整数或 0.1 网格。
- Cu Kalpha1 + Kalpha2；10--90 度 2theta；步长 0.02 度，共 4001 点。
- 固定 Rigaku MiniFlex-600-like 参考 profile，并加入分级的峰移、额外 Gaussian 展宽、
  非负平滑背景和 Poisson 计数噪声。
- 这些是 GSAS-II 生成的 benchmark 数据，不是真实仪器实测数据，也不应表述为某台仪器的测量。

## 返回文件

请优先返回填好的 `prediction_template.csv`。每个候选解中的每个预测相一行，并提供方法
原生数据库 ID、化学式和空间群；若某样本没有结果，也请保留一行并写明失败或超时原因。
